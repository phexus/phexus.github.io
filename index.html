Hello World

我的个人博客终于成立啦

这是我的第一篇博客，采用了 wsl + github pages + hexo 搭建而成。严格意义上讲，这是 hexo 自动生成的第一篇博客，正如学习程序程序设计语言都是先从 hello world 开始，这篇博客也正是以此为名。创建个人博客的目的也很简单，就是将平时学习到的点滴知识记录起来，聚少成多，不要让自己在同一个坑里掉两次。

或许，我早就应该成立这个博客了。不是很喜欢商业博客同意的风格，又加上平时用 evernote 满足了我大部分需求，这些都延缓了我的博客的诞生时间。我并不是一个排斥写作的人，这个博客除了记录一些知识外，大概也会记录记录自己的看法与生活吧。

嗯，争取做一个有温度的程序员。
